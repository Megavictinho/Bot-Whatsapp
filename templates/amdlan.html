AMDLAN / VSN

Listas de Automações

Automação por I.A

Automação por I.A quando o cliente seleciona o suporte remoto, a I.A verifica o que o cliente quer, e baseado nas frases cadastradas e seus respectivos níveis, a I.A coloca em um lugar diferente.
    ⊙ Automação para Suporte Remoto Normal / Urgente / Imediato.
        ⊛ Página Web para cadastro das Frases:http://vps57962.publiccloud.com.br/amdlanrisco

Automação por Dia

Automação baseada no dia da semana, selecionando automaticamente o cliente para setor Plantão.
    ⊙ Automação para Fim de Semana.
    ⊙ Automação para Feriado.
        ⊛ Página Web para cadastro dos Feriados: http://vps57962.publiccloud.com.br/amdlanferiado

Automação por Tipo

Automação por tipo específico de atendimento, como fila de prioridade ou atendimento em grupo.
    ⊙ Automação para Grupos de Whatsapp.
    ⊙ Automação para VIP.
        ⊛ Página Web para cadastro dos VIPs: http://vps57962.publiccloud.com.br/amdlanvip

Automação para Webhook

Automação baseada quando o técnico clica no botão de início de atendimento, o sistema salva em uma planilha para posteriormente poder calcular o tempo que o cliente esperou e o que o técnico fez no atendimento.
    ⊙ Automação para Webhook.
        ⊛ Página Web para Download da planilha: http://vps57962.publiccloud.com.br/amdlanplanilha

Automação para Sugestão de Vídeos

Automação que sugere vídeos para o cliente baseado em palavras-chave que ele possa digitar na sua solicitação.
    ⊙ Automação para Sugestão de Vídeo.
        ⊛ Página Web para cadastro de Vídeos: http://vps57962.publiccloud.com.br/amdlanvideo

Automação para Envio de Mensagem no Whatsapp

Automação que envia uma mensagem pré-determinada para o cliente quando o mesmo aprova uma proposta no Sistema Piperun.
    ⊙ Automação para Whatsapp.
        ⊛ Página Web Leitura do QR Code: http://vps57962.publiccloud.com.br:3000/dashboard

Automação para Atualização de Preços

Automação que atualiza de 04 em 04 horas os preços e estoques dos produtos no Piperun através do Keruak.
    ⊙ Automação para Atualização de Preços.
        ⊛ Clique abaixo para baixar o Log de Atualização de Preços:
            Baixar Log de Registro

Automação por Palavras-Chave

Automação por palavra-chave se caracteriza por, dependendo do que o cliente digitar, o sistema verifica se o texto possui alguma das palavras e o direciona corretamente.
    ⊙ Automação para XML.
    ⊙ Automação para Chave.
    ⊙ Automação para Suporte Presencial.
    ⊙ Automação para Suporte de Ponto.

Automação para Financeiro

Automação baseada no que o cliente digita no campo cnpjcpj, solicitado no início do atendimento. O sistema verifica se há algum boleto em aberto por mais de 10 dias.
    ⊙ Automação para Financeiro.
    ⊙ Automação para Envio de Boletos em Aberto para o Cliente.
    ⊙ Automação para Envio de orçamento direto para o funil de vendas do Piperun.

Automação para Técnico Interno

Automação baseada no que o Técnico digita no campo cnpjcpj (04.092.639/0001-03), solicitado no início do atendimento. O sistema verifica e o coloca em evidência para o suporte dar prioridade.
    ⊙ Automação para Técnico.

Aplicativo de Consulta Ao Keruak

Aplicativo que consulta o Keruak para verificar e adicionar informações, como Link, Chaves VPN. Com login em banco de dados online e log de acesso.
    ⊙ Aplicativo de Consulta Keruak para o suporte Remoto, sem informações de valores.
    ⊙ Aplicativo de Consulta Keruak para o comercial, com informações de valores.
